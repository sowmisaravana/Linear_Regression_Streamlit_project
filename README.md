 # Linear Regression Streamlit Project

This project is an interactive web application built using **Streamlit** that predicts outcomes based on user input data. It uses a **Linear Regression machine learning model** trained on a dataset to provide real-time predictions through a simple and user-friendly interface.

## Project Overview

The application allows users to enter input values through the Streamlit interface and instantly receive predictions generated by the trained Linear Regression model. This project demonstrates how machine learning models can be deployed into a web application for practical use.

## Features

- Interactive web interface using Streamlit
- Real-time predictions using Linear Regression
- Simple and clean user interface
- Easy to run locally
- Beginner-friendly machine learning deployment project

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Project Structure

Linear_Regression_Streamlit_Project

app.py – Main Streamlit application  
train_model.py – Script used to train the model  
dataset.csv – Dataset used for training  
model.pkl – Saved trained model  
requirements.txt – Required Python libraries  
README.md – Project documentation

## Installation

1. Clone the repository

git clone https://github.com/yourusername/Linear_Regression_Streamlit_Project.git

2. Navigate to the project directory

cd Linear_Regression_Streamlit_Project

3. Install the required dependencies

pip install -r requirements.txt

## Running the Application

Run the Streamlit app using the following command:

streamlit run app.py

After running the command, open your browser and go to:

http://localhost:8501

## How It Works

1. The dataset is used to train a Linear Regression model.
2. The trained model is saved as a file.
3. The Streamlit app loads the trained model.
4. Users enter input values in the web interface.
5. The model processes the input and displays the predicted result.
